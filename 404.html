React Remix Form Builder

A powerful, interactive form builder created with React, Remix, and Tailwind CSS.

Features:

  • Drag and drop interface for building custom forms
  • Multiple field types: text, number, select, checkbox, etc.
  • Form validation with real-time feedback
  • Light and dark mode support
  • Responsive design for all device sizes
